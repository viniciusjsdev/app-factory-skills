# Responsive QA Checklist

- [ ] No accidental page-level horizontal scroll at 360px
- [ ] Navigation remains reachable and does not cover content
- [ ] Forms and dialogs fit mobile width and height
- [ ] Tables use cards, column priority, or explicit overflow
- [ ] Charts retain readable labels and useful meaning
- [ ] Long content wraps without breaking layout
- [ ] Touch targets are comfortable
- [ ] Focus remains visible
- [ ] Reduced-motion preference is respected
- [ ] Layouts checked at 360, 390, 768, 1280, and 1440 widths
