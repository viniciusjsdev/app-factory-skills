# Interaction Checklist

Every important action must provide feedback.

Use:

- loading
- toast
- modal
- navigation
- state change
- confirmation
- disabled state
- localStorage persistence

Check:

- login works in mock mode
- logout works
- forms validate visually
- create flow works
- edit flow works when needed
- delete asks confirmation
- upload is visual when needed
- export/print works when needed
- copy action shows toast
- filters visibly affect list
- empty states exist
