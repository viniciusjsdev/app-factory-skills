# Quality Rubric

The Lovable prompt is good when it produces a prototype that feels:

- navigable
- complete
- specific to the domain
- visually coherent
- commercially presentable
- responsive
- rich in realistic data
- interactive
- ready for a demo
- prepared for future integration

The prompt is weak when it produces:

- generic dashboard
- empty pages
- dead buttons
- poor mobile behavior
- no mock persistence
- vague copy
- inconsistent visual style
- no main flow
- no detail screen
