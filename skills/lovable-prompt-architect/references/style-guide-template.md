# Style Guide Template

A Lovable prompt should define:

## Visual Tone

Examples:

- premium and institutional
- playful and mobile-first
- clean and professional
- technical and operational
- elegant and minimal
- bold and commercial

## Colors

Define primary, secondary, background, card, border, text, success, warning and danger colors when possible.

## Typography

Define font direction, title style, labels and text hierarchy.

## Components

Define card style, buttons, inputs, badges, modals, sidebars, nav, skeletons and empty states.

## Language

Define tone of copy and language.
