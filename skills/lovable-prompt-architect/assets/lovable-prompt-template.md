# RESPONSIVIDADE

A aplicacao deve funcionar bem em:

- desktop
- notebook
- tablet
- mobile

No mobile:

- cards devem empilhar
- menus devem adaptar
- tabelas devem virar cards ou ter scroll controlado
- formularios devem ser confortaveis
