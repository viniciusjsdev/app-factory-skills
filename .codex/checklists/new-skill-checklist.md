# New Skill Checklist

- Create `skills/<skill-name>/SKILL.md`.
- Add `skills/<skill-name>/agents/openai.yaml`.
- Add focused `references/` files only when needed.
- Add scripts only when they support repeatable validation.
- Keep the skill single-purpose.
- Add or update root docs/specs when the factory flow changes.
- Run `quick_validate.py` for the skill.
- Confirm no real secrets were added.
