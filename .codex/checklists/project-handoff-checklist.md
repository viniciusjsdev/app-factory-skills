# Project Handoff Checklist

- Product brief exists or assumptions are documented.
- Lovable prompt exists before Lovable generation.
- Frontend code exists before frontend normalization.
- Frontend architecture and API contract exist before backend work.
- Backend preflight context exists before backend files are created.
- Backend contract and env expectations exist before infra work.
- Infra validation was attempted before claiming local readiness.
- Remaining gaps are listed explicitly.
